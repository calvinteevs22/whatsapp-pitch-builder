UTILITY FUNCTIONS:
- fmt(): number formatting (B, M, K suffixes)
- fmtMoney(): currency formatting with abbreviations
- pct(): percentage formatting
- dm(): currency-aware display money function
- deriveAdv(): THE CORE CALCULATION ENGINE

CORE CALCULATION (deriveAdv):
Inputs: messages, deliveryRate, openRate, ctr, convRate, optOutRate, costPerMsg, dealValue
Calculates:
- delivered = messages * deliveryRate
- opened = delivered * openRate
- clicked = opened * ctr
- conversions = clicked * convRate
- revenue = conversions * dealValue
- spend = messages * costPerMsg
- roi = revenue/spend (if spend > 0)
- rev1k = (conversions/messages) * 1000 * dealValue
- cpConv = spend/conversions (cost per conversion)
- cpClick = spend/clicked (cost per click)
- reachRate = deliveryRate * openRate * 100
- moLost = messages * optOutRate
- yrLost = moLost * 12
- revAtRisk = yrLost * deliveryRate * openRate * ctr * convRate * dealValue

MAIN APP STRUCTURE:
- State: mode (basic/advanced), step (0-4), country, industry, compChannels, dealValue, chInputs, rTab, rRTab
- Input modes: volume (set messages) or goal (set target revenue, auto-calculate messages needed)
- Currency support: USD or local currency based on country selection
- Step wizard: Country → Industry → Inputs/Channels → Results
- Basic mode: 4 steps; Advanced mode: 5 steps (adds Channels step)
- Pre-built scenarios can be applied to auto-fill all inputs

RESULTS TABS:
- Tab 0: Scorecard (ScorecardTable + FunnelViz)
- Tab 1: Shift Simulator (ShiftSimulator)
- Tab 2: Audience Health (AudienceHealth)
- Tab 3: Break-Even (BreakEvenCard)
- Tab 4: Executive Summary (ExecSummary)
- Tab 5: Scenario Compare (ScenarioCompare)
- Tab 6: Monthly Projection (MonthlyProjection)

GOAL MODE:
- User sets target revenue instead of message volume
- Auto-calculates needed messages: Math.ceil(targetRevenue / revenuePerMsg)
- Shows: needed messages, spend, ROI, conversions
- Useful for reverse-engineering campaign size from revenue targets

ALL RESULTS COMPUTED:
- allR memo computes deriveAdv for each active channel
- Active channels: basic mode = [whatsapp, ...compChannels]; advanced = all with inputs
- Each channel gets full funnel metrics
